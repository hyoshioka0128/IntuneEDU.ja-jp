# <a name="default-ios-settings-in-express-configuration"></a>高速構成での既定の iOS 設定
高速構成は、デバイスまたはユーザーのグループをすばやくセットアップできるように、iOS 設定の提案を事前に設定しています。 Intune for Education は、Microsoft が推奨する設定と school 環境に最適な設定を選択します。 学校のルールやポリシーに合わせて変更を加えるか、または [設定] ページで [ストレート] をクリックして、推奨事項を適用します。 

設定と説明の完全な一覧については、「 [Intune for Education のすべての iOS 設定](all-edu-settings-ios.md)」を参照してください。 

> [!IMPORTANT]
> 高速構成を起動したときに iOS 構成が無効になっている場合は、MDM Apple Push certificate と DEP トークンの両方が設定されていることを確認してください。 これらの両方がある場合は、どちらも有効期限が切れていないことを確認します。 IOS デバイス管理のセットアップの詳細については、「 [ios デバイス管理のセットアップ](setup-ios-device-management.md)」を参照してください。


## <a name="app-store-itunes-store-and-book-store"></a>App Store、iTunes ストア、および書籍ストア  
設定|実行内容|  
|---|---|
|App Store をブロックする|学生が学校デバイスのアプリストアにアクセスできないようにします。|
|アプリ内購入をブロックする|実行中のアプリ内からストアの購入を試行します。|
|App Store と iTunes ストアの明示的なコンテンツをブロックする|学生がアプリストアでアダルトとして評価されたコンテンツにアクセスするのをブロックします。|
|アダルトとしてフラグが設定されている Apple ブックコンテンツのダウンロードをブロックする|学生がアダルトとして分類された書籍をダウンロードできないようにします。|  

## <a name="built-in-apps"></a>組み込みアプリ  
設定|実行内容|  
|---|---|
|カメラをブロックする|デバイスでカメラの使用をブロックします。|
|FaceTime をブロックする|デバイスでの FaceTime アプリの使用をブロックします。|
|ブロック Game Center|デバイスでの Game Center アプリの使用をブロックします。|  
|ブロック Apple Music|デバイスでの Apple Music アプリの音楽ストリーミングコンポーネントの使用をブロックします。|
|メッセージをブロックするアプリ|デバイスでのメッセージアプリの使用をブロックします。|
|Apple ブックをブロックする|学生が書籍ストアで書籍を閲覧したり購入したりすることを禁止します。|  

## <a name="device-restrictions"></a>デバイスの制限  
設定名|実行内容|
|---|---|
|デバイス名の変更をブロックする|学生がデバイスの名前を変更できないようにします。|
|壁紙の変更をブロックする|学生がデバイスのロック画面とホーム画面の画像を変更できないようにします。|
|通知設定の変更をブロックする|学生によるデバイス通知設定の変更をブロックします。|
|コンテンツとプライバシーの制限の変更をブロックする|学生がデバイスの制限 (保護者による制限) と画面時刻の設定を変更するのをブロックします。|
|すべてのコンテンツと設定を消去するブロックボタン|学生がデバイス上のすべてのコンテンツと設定を消去することをブロックします。 [消去] ボタンが使用できなくなり、選択できなくなります。|  

## <a name="icloud"></a>iCloud
|設定|実行内容|
|---|---|
|ICloud のバックアップをブロックする|学生がデバイスを iCloud にバックアップできないようにします。|
|ICloud へのドキュメントの同期をブロックします|ドキュメントから iCloud 記憶域スペースへの同期をブロックします。|
|ICloud フォトライブラリをブロックする|学生がクラウドに写真やビデオを保存できないようにします。 ICloud フォトライブラリから完全にダウンロードされていない写真は、ローカルストレージから削除されます。|  

## <a name="lock-screen-and-wallpaper"></a>画面と壁紙をロックする
設定|実行内容|
|---|---|
|ロック画面で通知をブロックする|デバイスがロックされているときに、学生が通知を表示できないようにします。|
|ロック画面からウォレットへのアクセスをブロックする|デバイスがロックされているときに、学生がウォレットアプリにアクセスすることをブロックします。|
|デバイスのロック画面の画像の設定|デバイスのロック画面の壁紙として表示するカスタムイメージを選択します。|
|デバイスのホーム画面イメージを設定する|デバイスのホーム画面の壁紙として表示するカスタムイメージを選択します。|  

## <a name="passcode-touch-id-and-face-id"></a>パスコード、タッチ ID、および顔 ID  
設定|実行内容|
|---|---|  
|パスコードが必要|デバイスのロックを解除するには、パスコードを入力するよう受講者に要求します。|
|パスコードの変更をブロックする|学生がデバイスのパスコードを変更、追加、または削除できないようにします。|
|デバイスをワイプするまでのパスコードの試行の失敗回数|ユーザーが許可されているサインイン試行回数を超えた場合、デバイスは、個人データ、iOS ソフトウェア、電子メールアカウント、システムとアプリの設定、ダウンロードされたアプリ、メディアなど、デバイスのすべてのコンテンツと設定を消去します。 デバイスは、初めてオンになったときの状態に復元されます。 この設定を構成するには、許可されるサインイン試行回数の最大値を入力します。|
|タッチ ID と顔 ID をブロックする|学生が指紋または顔認識を使用してデバイスのロックを解除することをブロックします。|

## <a name="siri-and-search"></a>Siri と検索 
設定|実行内容|
|---|---|   
|Siri をブロックする|学生が Siri (iOS 音声アシスタント) を使用できないようにします。|
